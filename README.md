# Penjelasan Program Ramadhan Countdown

Program ini adalah web aplikasi sederhana untuk menyambut Ramadhan 2026, menampilkan hitung mundur menuju Ramadhan, jam digital, dan jadwal Imsakiyah Kota Lubuklinggau.

## Fitur Utama
1. **Hitung Mundur Ramadhan**: Menampilkan waktu tersisa (hari, jam, menit, detik) menuju awal Ramadhan. Setelah waktu habis, tampilan berubah menjadi jam digital.
2. **Jam Digital**: Menampilkan waktu dan tanggal saat ini secara real-time.
3. **Jadwal Imsakiyah**: Menampilkan jadwal imsak, subuh, zuhur, ashar, maghrib, isya untuk setiap hari Ramadhan. Data jadwal diambil dari array JavaScript.
4. **Tombol Lihat Jadwal**: Pengguna dapat menampilkan atau menyembunyikan jadwal imsakiyah dengan tombol.
5. **Responsif**: Tampilan menyesuaikan untuk desktop dan mobile.
6. **Efek Visual**: Ada animasi bintang dan lentera untuk memperindah tampilan.

## Struktur File
- `index.html`: Struktur utama halaman, link ke CSS dan JS, elemen countdown, jam, tombol, dan jadwal.
- `css/style.css`: Mengatur tampilan, layout, animasi, dan responsivitas.
- `js/script.js`: Logika countdown, jam, pengelolaan jadwal, dan event tombol.

## Cara Kerja
- Saat halaman dibuka, countdown berjalan hingga Ramadhan dimulai.
- Setelah countdown selesai, jam digital tampil.
- Jadwal imsakiyah bisa ditampilkan/disebutkan dengan tombol.
- Semua tampilan dan data terupdate otomatis sesuai waktu.

Program ini cocok digunakan untuk menyambut Ramadhan, memberikan informasi waktu dan jadwal imsakiyah secara mudah dan menarik.
